## Description

This program maps old JSON medical cases into a new updated JSON structure.

## Usage

1. Make sure to place old versions in the local file 'cases'. 
   You can place JSON files directly in the file, or place ZIP files containing the medical cases.
2. Make sure python is installed on the device and run the command:
```python
python update_cases.py
```
3. Press enter to start loading.
4. New updated cases will be stored in the local file 'new_cases'.
